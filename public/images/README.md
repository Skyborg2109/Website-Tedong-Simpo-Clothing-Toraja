## 📸 CARA MENAMBAHKAN FOTO

### 1. Download dari Instagram

Buka: https://www.instagram.com/tedongsimpo.toraja/

**Untuk Logo:**
- Klik kanan pada profile picture
- Save as `logo.png`
- Upload ke folder ini

**Untuk Foto Produk:**
- Pilih foto produk terbaik
- Download menggunakan tools atau screenshot
- Save ke folder `products/`

### 2. Naming Convention

Gunakan nama yang deskriptif:

```
✅ BENAR:
- kaos-toraja-tedong-hitam.jpg
- hoodie-toraja-patedong-merah.jpg
- kaos-toraja-tongkonan-putih.jpg

❌ SALAH:
- IMG_1234.jpg
- photo.jpg
- product.jpg
```

### 3. Ukuran Rekomendasi

**Logo:**
- Navbar: 150x150px atau 200x200px
- Favicon: 32x32px, 16x16px
- Apple Touch: 180x180px

**Foto Produk:**
- Card view: 400x500px (portrait)
- Detail view: 800x1000px (high quality)
- Thumbnail: 150x150px (square)

### 4. Format File

- **Logo**: PNG (dengan transparent background lebih bagus)
- **Foto Produk**: JPG atau WEBP
- **Icons**: PNG atau SVG

### 5. Optimasi

**Compress gambar untuk loading cepat:**
- Target size: < 200KB per foto
- Tools: TinyPNG.com, Squoosh.app
- Format modern: WebP (optional)

## 🆘 TROUBLESHOOTING

### Gambar tidak muncul?
1. Cek nama file match dengan yang di code
2. Cek path folder sudah benar
3. Clear cache: `php artisan cache:clear`
4. Hard refresh browser: Ctrl + Shift + R

### Gambar terlalu besar?
1. Compress dengan TinyPNG.com
2. Resize ke ukuran yang sesuai
3. Convert ke WebP (optional)
