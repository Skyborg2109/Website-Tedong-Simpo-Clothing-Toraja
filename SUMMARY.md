## 🎨 CUSTOMIZATION GUIDE

### Mengubah Warna
Edit `public/css/style.css`:
```css
:root {
    --color-primary: #1a1a1a;
    --color-secondary: #C41E3A;
    --color-accent: #D4AF37;
    --color-wood: #8B4513;
}
```

### Menambah Produk
Edit `resources/views/products/index.blade.php`
Copy-paste product card dan sesuaikan informasi

### Mengubah Kontak
Edit `resources/views/layouts/app.blade.php`
Cari section footer dan update informasi
